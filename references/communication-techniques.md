# 家校沟通技巧参考

综合名班主任工作室实践与沟通心理学方法，供生成草稿时选用。**教师须根据具体家庭情境调整。**

## 沟通四原则

1. **尊重真诚、平等结盟** — 语气语调体现合作，而非居高临下
2. **事实说话、传递师爱** — 「录像机语言」：只描述可观察行为，不评判人格
3. **换位思考、理解赋能** — 先理解家长焦虑，再共谋策略
4. **明确边界、积极引导** — 分清学校与家庭角色，看见动力点

## 沟通三阶段

| 阶段 | 要点 | 示例动作 |
|------|------|----------|
| **沟通前** | 聚焦诉求、准备材料、明确目标 | 整理事实记录；想清楚希望家长配合什么 |
| **沟通中** | 真诚+互动；一次只解决一个核心问题 | 先听再说；用具体事例替代模糊评价 |
| **沟通后** | 及时跟进、反馈成长动态 | 记录沟通要点；约定下次跟进节点 |

## 三明治反馈法（夹心面包）

适用于需指出改进点的沟通：

```
1. 认同/赞赏 — 先肯定优点或努力
2. 建议/改进 — 陈述具体观察与配合事项
3. 鼓励/期待 — 表达信任与合作意愿
```

**示例**：
> 孩子思维活跃，小组讨论参与度较高（肯定）。最近几次作业书写规范性还需要加强（建议）。我们可以一起帮他制定整理书包的小流程，我相信他会慢慢稳定下来（鼓励）。

## 录像机语言 vs 评价语言

| 避免（评价/标签） | 改用（录像机/事实） |
|-------------------|---------------------|
| 孩子上课总走神 | 今天数学课前 15 分钟，有三次低头玩笔 |
| 孩子总是忘带作业 | 最近两周有 3 次未带作业本 |
| 家长没有督促 | 想请您协助孩子养成每晚整理书包的习惯 |
| 孩子太差/不认真 | 课前准备还不够稳定，我们在班级也做了提醒 |

## 非暴力沟通框架（敏感场景）

```
观察（事实）→ 感受（共情）→ 需要（共同目标）→ 请求（具体配合）
```

**示例**：
> 我注意到您对孩子成绩下滑很担心（观察+感受）。我们的目标都是帮助孩子稳定下来（需要）。您想听听我这边观察到的具体细节吗？（请求）

## 不同家长类型的调整

| 类型 | 沟通侧重 | 语言提示 |
|------|----------|----------|
| 祖辈家长 | 健康、安全、快乐 | 亲切口语；少用术语；「咱们家孩子」「咱们一起」 |
| 焦虑型家长 | 先共情再方案 | 「我理解您的担心」→ 具体事实 → 可操作的配合点 |
| 忙碌型家长 | 短、清、有截止 | 分条列述；关键时间加粗；一条消息一个主题 |
| 特殊家庭 | 分寸、尊重 | 避免比较；不暗示家庭原因；私下沟通 |

## 常用开场与过渡语

- 「家长您好，想和您沟通一下孩子近期……的情况。」
- 「先跟您同步一下本周班级的情况……」
- 「我们这边也会继续在学校提醒，一起帮助孩子……」
- 「如有疑问，欢迎随时跟我联系。」

## 一次只谈一个核心问题

家长同时提多个问题时，引导聚焦：

> 「您提到的作业、态度和纪律都很重要。咱们今天先一起商量『如何制定每周作业计划』，其他问题下次再细聊，好吗？」
