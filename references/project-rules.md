# 班主任 AI 工作台规则

本项目用于辅助班主任处理通知、班级总结、学生评语和家校沟通等文字任务。

输出对象可能包括家长、学生、学校内部人员和教师本人。

## 基本原则

- 对外内容必须准确、清楚、语气平和
- 不编造学校没有明确说明的信息
- 不直接暴露学生个人隐私、成绩、家庭、健康等敏感信息
- 不对学生做标签化判断
- 涉及家校沟通、学生评价、冲突处理、安全健康等内容时，必须提醒教师人工复核
- AI 只生成初稿和整理建议，最终判断由教师负责

## 资料存放约定

| 目录 | 用途 |
|------|------|
| `01-长期资料库/` | 稳定参考：模板、规范、样例、班级背景 |
| `02-临时输入/` | 当前任务材料（沟通素材须脱敏） |
| `04-任务记录/` | 每次任务的过程记录 |
| `05-输出结果/` | 初稿及审核后版本 |

## 输出规范

每份对外初稿末尾必须包含：

```markdown
## 教师复核清单
- [ ] …
- [ ] …

## 表达风险提醒（如有）
- …
```
