# 班级周总结

## 任务重点

**结构和取舍**。一周事件很多，不是全部适合发给家长。AI 整理初稿，**教师判断哪些适合公开**。

## 执行前读取

- `01-长期资料库/输出模板/班级周总结模板.md`
- `01-长期资料库/班级背景/班级整体说明.md`
- `01-长期资料库/学校要求/家校沟通规范.md`
- `02-临时输入/本周记录/` 中的本周记录
- `01-长期资料库/表达边界/班主任表达边界.md`

## 执行步骤

1. **通读本周记录**，标注：适合公开 / 仅内部 / 需脱敏后单独沟通
2. **分类整理**：整体情况、亮点、需提醒事项、下周安排
3. **生成家长版初稿**：不点名、提醒式表达（非指责式）
4. **提取可流转信息**：需家长配合的事项 → 可生成通知或沟通草稿
5. **输出**并附复核清单

## 输出格式

```markdown
# 第N周班级周报初稿

各位家长好：

【本周情况】
…

【值得肯定】
…

【需要提醒】
…

【下周安排】
…

---
## 教师复核清单
- [ ] 是否有不适合公开的班级内部信息
- [ ] 活动/时间是否与学校安排一致
- [ ] 是否有点名或指责式表达
- [ ] 需单独沟通的事项是否已从周报中剥离

## 可提取为后续任务
- 通知事项：…
- 个别沟通（脱敏）：…
```

## 表达要求

- 共性问题用「部分同学」「个别孩子」
- 问题描述用「还需要继续提醒/加强」而非「表现很差」
- 下周安排注明「以学校后续通知为准」（如尚未最终确认）
