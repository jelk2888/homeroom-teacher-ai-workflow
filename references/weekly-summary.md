# 班级周总结

## 任务重点

**结构和取舍。** 一周事件很多，不是全部适合发给家长。AI 整理初稿，**教师判断哪些适合公开**。

## 周报五要素

一份完整的家长周报应包含：

| 要素 | 内容 | 示例 |
|------|------|------|
| **本周学习要点** | 各学科重点内容 | "本周语文重点学习了……" |
| **学生表现** | 整体情况 + 个别关注（不点名） | "大部分同学在……方面表现突出" |
| **值得肯定** | 具体亮点和进步 | "班级卫生情况较前期有所进步" |
| **需要提醒** | 共性问题的温和提醒 | "部分同学出现美术材料准备不充分的情况" |
| **下周安排** | 学习计划 + 活动安排 + 家长配合事项 | "下周五学校将举行科技节展示活动" |

## 执行步骤

1. **通读本周记录**，标注每条信息的公开级别：
   - ✅ 适合公开（班级整体情况、活动安排、共性亮点）
   - ⚠️ 需脱敏后公开（个别学生问题 → 转为共性提醒）
   - ❌ 仅内部记录（点名批评、敏感事件、学生隐私）
2. **分类整理**：本周情况 → 值得肯定 → 需要提醒 → 下周安排
3. **生成家长版初稿**：不点名、提醒式表达（非指责式）
4. **提取可流转信息**：需家长配合的事项 → 可生成通知或沟通草稿
5. **输出**并附复核清单

## 输出格式

```markdown
# 第N周班级周报初稿

各位家长好：

[开场：本周整体状态概括，1-2句]

【本周情况】
[班级整体表现，教学和活动重点]
…

【值得肯定】
[亮点和进步，具体但匿名]
…

【需要提醒】
[共性问题的温和提醒，附具体建议]
…

【下周安排】
[下周重点+家长配合事项]
…

---
## 教师复核清单
- [ ] 是否有不适合公开的班级内部信息
- [ ] 活动/时间是否与学校安排一致
- [ ] 是否有点名或指责式表达
- [ ] 需单独沟通的事项是否已从周报中剥离

## 可提取为后续任务
- 通知事项：…
- 个别沟通（脱敏）：…
```

## 话术要求

### 表扬话术
```
[学生群体/班级整体] 本周在 [方面] 进步明显，[具体表现]，值得肯定！
```

### 提醒话术
```
[学生群体/个别情况] 在 [方面] 还需要继续关注，建议家长 [具体建议]。
```

### 共性问题表达
- ✅ "部分同学""个别孩子""多数同学"
- ❌ 不点名、不暗示具体学生

## 表达要求

- 问题描述用「还需要继续提醒/加强」而非「表现很差」
- 共性问题用「部分同学」「个别孩子」
- 下周安排注明「以学校后续通知为准」（如尚未最终确认）
- 开场保持积极基调，不以上周遗留问题开篇
- 结尾感谢家长配合，保持合作姿态
