# 家长通知改写

## 任务重点

**准确和清楚**。学校原通知中的时间、地点、对象、材料、截止时间必须保留；可改表达使其更适合家长阅读，**不可补充原文没有的信息**。

## 执行前读取

- `01-长期资料库/学校要求/` 中的家校沟通规范
- `01-长期资料库/输出模板/家长通知模板.md`
- `01-长期资料库/过往样例/` 中优秀通知样例
- `02-临时输入/学校通知/` 中的原始通知

## 执行步骤

1. **提取必保留信息**：时间、地点、对象、材料、截止时间、联系方式
2. **对照原文**：列出原文有而初稿不可遗漏的条目
3. **按模板改写**：语气平和、段落清晰、重点前置
4. **自检**：是否添加了原文没有的内容？是否改变了原意？
5. **输出**到 `05-输出结果/` 并附复核清单

## 输出格式

```markdown
# [通知主题] 家长通知初稿

各位家长好：

【通知事项】
…

【时间地点】
- 时间：
- 地点：

【需要准备】
…

【家长需要配合】
…

【温馨提醒】
…

---
## 教师复核清单
- [ ] 时间地点是否与学校原文一致
- [ ] 是否补充了原文没有的信息
- [ ] 截止时间和材料清单是否完整
```

## 禁止事项

- 自行补充活动细节、费用、承诺
- 把「以学校通知为准」类事项写成确定安排
- 使用恐吓式或过度催促语气
