# 家长通知改写

## 任务重点

**准确和清楚**。学校原通知中的时间、地点、对象、材料、截止时间必须保留；可改表达使其更适合家长阅读，**不可补充原文没有的信息**。

## 执行前读取

- `01-长期资料库/学校要求/` 中的家校沟通规范
- `01-长期资料库/输出模板/家长通知模板.md`
- `01-长期资料库/过往样例/` 中优秀通知样例
- `02-临时输入/学校通知/` 中的原始通知

## 执行步骤

1. **提取必保留信息**：时间、地点、对象、材料、截止时间、联系方式
2. **对照原文**：列出原文有而初稿不可遗漏的条目
3. **按模板改写**：语气平和、段落清晰、重点前置
4. **自检**：是否添加了原文没有的内容？是否改变了原意？
5. **输出**到 `05-输出结果/` 并附复核清单

## 输出格式

```markdown
# [通知主题] 家长通知初稿

各位家长好：

【通知事项】
…

【时间地点】
- 时间：
- 地点：

【需要准备】
…

【家长需要配合】
…

【温馨提醒】
…

---
## 教师复核清单
- [ ] 时间地点是否与学校原文一致
- [ ] 是否补充了原文没有的信息
- [ ] 截止时间和材料清单是否完整
```

## 标准结构（通知写作通则）

```
标题/主题 → 称呼 → 正文（时间→地点→事项→要求）→ 温馨提醒 → 落款/日期
```

正文须说清**时间、地点、人物/对象、事件、要求**五要素；事项多时用分条列述。

## 八大常见误区（生成后自检）

1. **信息过载** — 单条通知超过 5 个事项 → 拆条或分次发
2. **表述模糊** — 「近期」「尽快」→ 改为具体日期时间
3. **专业术语** — 教务系统内部用语 → 改为家长能懂的话
4. **格式混乱** — 中英文符号混用 → 统一中文标点
5. **隐私风险** — 群发显示手机号、点名批评 → 脱敏处理
6. **重复发送** — 同一信息多次推送 → 合并或注明「以此为准」
7. **过期内容** — 时间已过仍发送 → 核对日期
8. **擅自补充** — 原文没有的费用/承诺 → 删除或标注「以学校通知为准」

## 群公告格式建议

- 标题：【X班】+ 主题
- 关键时间、地点、截止事项单独成行
- 附件说明获取方式（群文件/链接/有效期）
- 日常通知宜固定时段发布，每周 ≤3 次

## 禁止事项

- 自行补充活动细节、费用、承诺
- 把「以学校通知为准」类事项写成确定安排
- 使用恐吓式或过度催促语气
- 在群通知中点名批评个别学生
