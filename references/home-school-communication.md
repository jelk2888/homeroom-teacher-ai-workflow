# 家校沟通草稿

## 任务重点

**语气和边界**。涉及具体学生和家长，**最不可直接生成后就发送**。先整理事实，区分可说/不可说，再由教师判断语气。

## 执行前读取

- `01-长期资料库/学校要求/家校沟通规范.md`
- `01-长期资料库/表达边界/班主任表达边界.md`
- `01-长期资料库/输出模板/家校沟通草稿模板.md`
- `02-临时输入/家校沟通素材/` 中的**脱敏**事实记录

## 执行步骤

1. **整理事实**：时间、次数、具体行为（不含标签化评价）
2. **明确沟通目标**：希望家长配合什么、教师已做什么
3. **生成草稿**：先说事实 → 教师态度 → 配合事项 → 后续跟进
4. **列出表达风险**：不宜直接发送的表述及原因
5. **输出**并附复核清单

## 输出格式

```markdown
# 家校沟通草稿

家长您好，…

[事实说明段落]

[教师态度与已做工作]

[希望家长配合的具体事项]

[后续跟进建议]

---
## 表达风险提醒
- 不建议写「…」，因为…
- 如需提到具体次数，请教师确认记录是否准确

## 教师复核清单
- [ ] 事实记录是否准确
- [ ] 语气是否适合该家庭之前的沟通基础
- [ ] 是否有指责家长或标签化学生的表述
- [ ] 敏感信息是否已脱敏
```

## 语气原则

- 平和、合作、具体
- 用「最近两周有几次…」替代「总是…」
- 用「想请您协助…」替代「您没有督促…」
- 说明教师端也会继续跟进，形成合力

## 禁止事项

- 在未获教师确认前建议「直接发送」
- 扩大问题或暗示家庭原因
- 替学校承诺处理方式
