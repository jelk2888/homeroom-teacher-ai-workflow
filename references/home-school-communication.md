# 家校沟通草稿

## 任务重点

**语气和边界。** 涉及具体学生和家长，**最不可直接生成后就发送**。先整理事实，区分可说/不可说，再由教师判断语气。

## 沟通四大原则

| 原则 | 做法 |
|------|------|
| **真诚共情** | 站在家长视角理解焦虑与期待，用生活化表达拉近距离 |
| **客观精准** | 以事实、数据、具体行为描述为依据，避免模糊评价 |
| **目标导向** | 明确沟通目的，给出可操作的行动建议 |
| **隐私保护** | 涉及个体内容适度模糊化，群体反馈侧重共性特征 |

## 五类沟通场景

| 场景 | 重点 | 示例话术引导 |
|------|------|-------------|
| **日常关怀** | 传递温度，拉近距离 | 季节问候、班级趣事分享、进步表扬 |
| **学业反馈** | 优点+问题+具体建议 | 用数据和具体表现，避免笼统评价 |
| **行为习惯提醒** | 合作化表达，不指责 | "想请您协助孩子……" |
| **活动通知配合** | 说清要求，降低家长负担 | 明确配合事项和截止时间 |
| **紧急/特殊情况** | 及时+冷静+方案 | 先处理再沟通，事实+影响+方案 |

## 执行步骤

1. **整理事实**：时间、次数、具体行为（不含标签化评价）
2. **判断场景**：属于关怀/反馈/提醒/通知/紧急哪种类型
3. **明确沟通目标**：希望家长配合什么、教师已做什么
4. **生成草稿**：
   - 先说事实 → 教师态度 → 配合事项 → 后续跟进
   - 语气：平和、合作、具体
5. **列出表达风险**：不宜直接发送的表述及原因
6. **输出**并附复核清单

## 输出格式

```markdown
# 家校沟通草稿

[对象]家长您好，…

[事实说明段落]

[教师态度与已做工作]

[希望家长配合的具体事项]

[后续跟进建议]

---
## 表达风险提醒
- 不建议写「…」，因为…
- 如需提到具体次数，请教师确认记录是否准确

## 教师复核清单
- [ ] 事实记录是否准确
- [ ] 语气是否适合该家庭之前的沟通基础
- [ ] 是否有指责家长或标签化学生的表述
- [ ] 敏感信息是否已脱敏
- [ ] 是否需要在发送前电话预沟通（敏感话题）
```

## 语气原则

- 平和、合作、具体
- 用「最近两周有几次…」替代「总是…」
- 用「想请您协助…」替代「您没有督促…」
- 说明教师端也会继续跟进，形成合力

## 沟通时效参考

| 事项 | 建议时效 |
|------|----------|
| 正向反馈（进步表扬） | 当天内通知，增强学生信心 |
| 一般行为习惯提醒 | 发觉后1-2个工作日内联系 |
| 学业问题反馈 | 周末或工作日上午发送 |
| 紧急安全事项 | 即时电话沟通 |
| 学生冲突/受伤 | 2小时内联系家长 |

## 禁止事项

- 在未获教师确认前建议「直接发送」
- 扩大问题或暗示家庭原因
- 替学校承诺处理方式
- 使用「总是」「从来不」「太差」等绝对化/标签化词语
- 深夜或凌晨发送非紧急消息
