# 班主任表达边界

> 参考：教育部《教师生成式人工智能应用指引（第一版）》规范指引部分（"约束清单"）
> AI 只生成初稿和整理建议，最终判断由教师负责。

## 对家长表达时

- 先说事实，再说建议，避免直接评价家长
- 避免使用「总是」「从不」「太差」「不认真」等绝对化词语
- 涉及学生问题时，尽量描述具体行为，不给学生贴标签
- 涉及成绩、健康、家庭情况时，必须脱敏并提醒人工复核
- 不替学校作出原通知没有写明的承诺
- 不把内部管理要求直接转成对家长的指责

## 家校沟通四大原则

| 原则 | 说明 | 示例 |
|------|------|------|
| **真诚共情** | 站在家长视角理解教育焦虑与期待，用生活化表达拉近距离 | "我注意到孩子最近……我们一起看看怎么帮他" |
| **客观精准** | 以事实、数据、具体行为描述为依据，避免模糊评价或主观臆断 | "最近两周有3次忘带作业本"而非"总是不带作业" |
| **目标导向** | 明确沟通目的（反馈/协作/关怀），给出可操作的行动建议 | "想请您协助孩子每天晚上按课程表整理书包" |
| **隐私保护** | 涉及个体内容适度模糊化，群体反馈侧重共性特征 | 群发用"部分同学""个别孩子"，不点名 |

## 四类任务边界速查

| 任务 | 可以做 | 不可以做 |
|------|--------|----------|
| 通知 | 改写表达、调整结构、补充温馨提醒 | 补充原文没有的时间/地点/要求/费用/承诺 |
| 周总结 | 整理亮点与提醒、分类归纳 | 点名批评、公开不宜公开的细节 |
| 评语 | 基于记录组织语言、具体描述行为 | 编造表现、标签化、千篇一律空话、把阶段性问题写成永久缺陷 |
| 家校沟通 | 整理事实+配合建议、合作化表达 | 指责家长、过度承诺、扩大问题、暗示家庭原因 |

## 敏感信息处理

以下内容不得出现在对外输出中（除非教师明确授权且已脱敏）：

- 学生全名（群发场景用「部分同学」「个别孩子」等）
- 具体分数、排名
- 家庭结构、经济状况
- 健康、心理、处分记录

## 时间与渠道边界

| 事项 | 建议 |
|------|------|
| 紧急事项 | 即时电话或当面沟通 |
| 一般问题反馈 | 24小时内启动沟通程序 |
| 非紧急通知 | 避免深夜或凌晨发送 |
| 学业反馈 | 建议周末或工作日上午发送 |
| 问题类沟通 | 发觉后1-2个工作日内联系，避免问题积累 |
| 沟通渠道 | 私密事项优先私信，公开事项走班级群 |

## 语气转换对照表

| 不推荐 | 推荐 |
|--------|------|
| "您的孩子总是……" | "最近观察到孩子有几次……" |
| "您没有督促……" | "想请您协助孩子……" |
| "表现很差" | "还需要继续提醒和加强" |
| "太不像话了" | "这个行为还需要引导" |
| "别的孩子都能做到" | "每个孩子的节奏不一样，我们一起帮助TA" |
| "我保证……" | "我会持续关注，及时和您沟通" |
| "这是家长的失职" | "家校配合对孩子成长很重要" |

## 教师 AI 使用底线（教育部指引相关）

- AI 生成的内容，教师必须逐条审查后才能对外发送
- 涉及学生评价、家校沟通、冲突处理的内容，不得未经教师复核直接使用
- 不得使用 AI 编造或推测学生的家庭背景、心理状态
- AI 辅助生成的材料如涉及个人数据，须脱敏处理后再交由 AI 处理
- 使用 AI 生成评语时，必须逐个学生核对输出内容的准确性，不得批量导入后直接使用
