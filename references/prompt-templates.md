# AI 提示词模板集

> 将 `[ ]` 中的占位符替换为你的实际情况即可使用。

## 提示词黄金结构

所有高效提示词遵循这个底层逻辑：

```
角色 + 任务 + 要求 + 格式 + 示例（可选）
```

- **角色**：让 AI 代入专业身份（如"你是一位资深班主任"）
- **任务**：明确要它做什么
- **要求**：语气、长度、风格、禁忌等
- **格式**：输出结构（段落、列表、表格等）
- **示例**：给1~2个例子，让 AI 模仿输出风格

## 一、班级周总结

### 模板：从记录生成周报

```
你是一位经验丰富的班主任。请基于以下本周班级记录，生成一份适合发给家长的班级周报初稿。

本周记录：
[粘贴 02-临时输入/本周记录/ 中的内容]

要求：
- 分「本周情况」「值得肯定」「需要提醒」「下周安排」四个板块
- 不点名学生，共性问题用"部分同学""个别孩子"
- 对问题使用提醒式表达，不使用指责式表达
- 语气平和、积极
- 末尾附教师复核清单

参考模板：
[粘贴 班级周总结模板.md 的内容]
```

### 模板：快速周报（极简版）

```
你是一位班主任。请整理以下班级记录，生成200字以内的家长周报摘要：

记录：
[粘贴简要记录]

要求：只说亮点和1个温和提醒，语气亲切。
```

## 二、家长通知改写

### 模板：标准通知改写

```
你是一位班主任。请将以下学校通知改写为适合发给家长的通知：

学校原通知：
[粘贴学校通知原文]

要求：
- 必须保留原文中的时间、地点、对象、材料、截止时间
- 不补充原文没有的信息（不添加费用、承诺、活动意义等）
- 语言平和、清楚，适合家长阅读
- 使用模板格式：「通知事项」「时间地点」「需要准备」「家长需要配合」「温馨提醒」
- 末尾附教师复核清单
```

### 模板：紧急通知简化

```
你是一位班主任。请将以下学校紧急通知精炼为适合班级群发送的3句话通知：

原通知：[粘贴]

要求：核心信息（时间+地点+行动）前置，语气冷静但不制造恐慌。
```

## 三、学生评语初稿

### 模板：单个学生评语

```
你是一位班主任。请基于以下学生表现记录，生成一份期末评语初稿。

学生类型：[优异/进步中/有特长/内向/需改进]
学段：[小学低段/小学中高段/初中/高中]

表现记录：
[粘贴已脱敏的表现记录]

要求：
- 只根据记录中的事实表达，不编造表现
- 语气积极、具体、平衡（优点+成长建议并重）
- 避免标签化和绝对化判断（不用"总是""太差"等）
- 表扬要具体到事件，不足用"如果能……就更好了"表达
- 末尾列教师需要核对的事实点和复核清单
```

### 模板：五维评语法

```
你是一位班主任。请从以下五个维度为学生撰写评语初稿，每个维度1-2句话：
1. 学业成绩（过程分析，不只看分数）
2. 学习态度（行为追踪，不笼统评价）
3. 品德素养（事件锚定，不空泛赞美）
4. 个性发展（特色挖掘，不千篇一律）
5. 师生/家校互动（协作呈现，不单向评价）

学生记录：
[粘贴记录]

要求：合成一段通顺的评语，150-200字，积极具体，附复核清单。
```

## 四、家校沟通草稿

### 模板：行为习惯沟通

```
你是一位班主任。请基于以下事实记录，生成一份适合私信家长的家校沟通草稿。

事实记录：
[粘贴脱敏后的事实记录]

要求：
- 先整理事实（时间、次数、具体行为）
- 语气平和、合作，不指责学生和家长
- 使用"想请您协助……"而非"您没有……"
- 明确希望家长配合的具体事项
- 说明教师端已做和将做的工作
- 末尾列出表达风险提醒（哪些表述不宜发送及原因）
```

### 模板：进步表扬沟通

```
你是一位班主任。学生[代号]最近在[方面]有明显进步：
[具体表现]

请生成一段发给家长的表扬私信，要求：
- 具体描述进步（不空泛说"表现好"）
- 语气真诚温暖
- 激励家长继续配合
- 50-100字
```

### 模板：活动配合通知

```
你是一位班主任。学校将举行[活动名称]，需要家长配合以下事项：
[配合事项列表]

请生成一段班级群通知，要求：
- 活动信息简洁清晰（时间+地点+需准备什么）
- 配合事项用"请家长协助……"的温和表达
- 附一句温馨提醒（如天气/安全/鼓励孩子）
- 150字以内
```

## 五、日常工作速查模板

### 班级群规则发布

```
你是一位班主任。请为[年级]班级群撰写群规则公告，包含：
1. 群用途说明（通知发布、活动分享）
2. 消息阅读提醒
3. 私聊建议（个人问题请私信老师）
4. 紧急联系说明（紧急事件请电话）
要求：语气亲切但不失专业，150字以内。
```

### 学期初收心通知

```
你是一位班主任。请撰写一段开学前"收心提醒"发到班级群，包含：
1. 开学时间和报到安排
2. 作业检查提醒
3. 作息调整建议
4. 学习用品准备提醒
要求：语气温馨鼓励，200字以内。
```

### 考前温馨提示

```
你是一位班主任。请撰写一段考试前发给家长的温馨提示，包含：
1. 考试时间和科目安排
2. 文具准备提醒
3. 作息和饮食建议
4. 心态调整鼓励
要求：不制造焦虑，语气平和鼓励，150字以内。
```

## 使用建议

1. **先建资料库再提问**：把模板、规范、样例放进 `01-长期资料库/` 后，提示词只需引用路径
2. **复杂任务分步走**：不要一次让 AI 写完全部，先整理事实 → 再生成初稿 → 最后复核
3. **带上"约束清单"**：每次提问都在要求中加入"不点名""不标签化""附复核清单"
4. **积累优秀样例**：把教师修改后满意的输出存进 `过往样例/`，下次让 AI 参考
