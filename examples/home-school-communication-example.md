# 示例：从事实记录生成家校沟通草稿

## 输入材料

假设 `02-临时输入/家校沟通素材/学习用品准备情况.md` 内容如下（已脱敏）：

```markdown
# 家校沟通事实记录

- 学生最近两周有 3 次忘带作业本。
- 课堂上能参与活动，但课前准备不够稳定。
- 已在班级内提醒整理书包和检查作业。
- 希望家长协助孩子每天晚上按课程表整理学习用品。
```

## Trae 调用方式

```
请使用 03-Skills/家校沟通草稿/SKILL.md。

参考：
- 01-长期资料库/学校要求/家校沟通规范.md
- 01-长期资料库/表达边界/班主任表达边界.md
- 01-长期资料库/输出模板/家校沟通草稿模板.md

处理：
- 02-临时输入/家校沟通素材/学习用品准备情况.md

要求：
- 先整理事实。
- 再生成一版适合私信家长的沟通草稿。
- 语气平和，不指责学生和家长。
- 明确希望家长配合的具体事项。
- 最后列出不宜直接发送的表达风险。
```

## 输出示例

```markdown
# 家校沟通草稿

家长您好，最近两周观察到孩子有几次忘带作业本的情况，在班级里我们也提醒过整理书包和检查作业的方法。孩子课堂参与度还不错，就是课前准备还需要再稳定一些。想请您协助孩子每天晚上按课程表整理学习用品，慢慢养成习惯就好。我这边也会继续关注，有什么情况再及时和您沟通。谢谢您的配合！

---

## 表达风险提醒
- 不建议写「孩子总是忘带作业」，因为"总是"是绝对化表述，且可能让家长感到被指责。
- 不建议写「您需要在家庭教育中加强督促」，因为这样的话暗示家庭原因，超出了教师可以判断的范围。
- 如需提到"3次"的具体次数，请教师确认记录是否准确。

## 教师复核清单
- [ ] 事实记录是否准确（次数、时间）
- [ ] 语气是否适合该家庭之前的沟通基础
- [ ] 是否有指责家长或标签化学生的表述
- [ ] 是否需要根据与家长的历史沟通调整措辞
```

## 关键要点

- 先说事实，再说教师态度和在班级已做的工作
- 用"协助"而非"督促"，语气合作化
- 不把问题扩大为对家庭教育的暗示
- 表达风险提醒让教师明白哪些表述不宜发送
- 最终发送前必须教师人工审查
