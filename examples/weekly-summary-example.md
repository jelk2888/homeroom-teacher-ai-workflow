# 示例：从一周班级情况生成班级周报

## 输入材料

假设 `02-临时输入/本周记录/第12周班级记录.md` 内容如下：

```markdown
# 本周班级记录

1. 周一升旗仪式整体秩序较好，但入场速度偏慢。
2. 周三进行了安全教育主题班会，学生参与度较高。
3. 本周有部分学生忘带美术材料，影响课堂准备。
4. 班级卫生整体有进步，值日小组交接还需要更明确。
5. 下周五学校有科技节展示活动，需要家长提醒孩子带好作品材料。
```

## Trae 调用方式

```
请使用 03-Skills/班级周总结/SKILL.md。

参考：
- 01-长期资料库/输出模板/班级周总结模板.md
- 01-长期资料库/班级背景/班级整体说明.md
- 01-长期资料库/学校要求/家校沟通规范.md

处理：
- 02-临时输入/本周记录/第12周班级记录.md

要求：
- 输出一版适合发给家长的班级周报初稿。
- 不点名学生。
- 对问题使用提醒式表达，不使用指责式表达。
- 最后列出教师复核清单。
```

## 输出示例

```markdown
# 第12周班级周报初稿

各位家长好：

本周班级整体状态平稳，孩子们在集体活动、主题班会和班级卫生方面都有不少进步。

【本周情况】
本周升旗仪式中，同学们整体秩序较好，能够按照要求参与集体活动。周三班级开展了安全教育主题班会，多数同学能积极参与讨论，安全意识有进一步提升。

【值得肯定】
本周班级卫生情况较前期有所进步，值日同学能够主动完成基本任务，班级环境保持得更整洁。

【需要提醒】
本周有部分同学出现美术材料准备不充分的情况，影响了课堂活动开展。请家长帮助孩子提前查看课程安排，逐步养成自己整理学习用品的习惯。

【下周安排】
下周五学校将举行科技节展示活动，请家长提醒孩子按要求带好作品材料，具体要求以学校后续通知为准。

教师复核清单：
- 请确认科技节时间是否仍为下周五。
- 请确认是否需要补充作品材料的具体清单。
- 请确认周报中是否有不适合公开表达的班级内部信息。
```

## 关键要点

- 不点名 → 用"部分同学""多数同学"
- 问题用"提醒"而非"批评"→ "需要提醒"而非"表现不好"
- 活动信息注明"以学校后续通知为准"
- 复核清单让教师最终把关
