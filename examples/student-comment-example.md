# 示例：从学生表现生成期末评语初稿

## 输入材料

假设 `02-临时输入/评语素材/学生A阶段性表现记录.md` 内容如下（已脱敏）：

```markdown
# 学生A阶段性表现记录

- 课堂上愿意回答问题，尤其在小组讨论中参与度较高。
- 作业能按时完成，但书写规范性还需要继续提醒。
- 本学期担任过小组记录员，能完成基本记录任务。
- 遇到难题时有时会急躁，需要继续练习整理思路。
- 与同学相处总体融洽，愿意帮助同伴。
```

## Trae 调用方式

```
请使用 03-Skills/学生评语初稿/SKILL.md。

参考：
- 01-长期资料库/输出模板/学生评语模板.md
- 01-长期资料库/过往样例/评语样例.md
- 01-长期资料库/学校要求/评语表达规范.md

处理：
- 02-临时输入/评语素材/学生A阶段性表现记录.md

要求：
- 生成一版期末评语初稿。
- 只根据记录中的事实表达，不编造表现。
- 语气积极、具体、平衡。
- 避免标签化和绝对化判断。
- 最后列出教师需要核对的事实点。
```

## 输出示例

```markdown
# 学生A期末评语初稿

你在本学期的学习和集体生活中表现出了较好的参与意识。课堂上，你愿意表达自己的想法，在小组讨论中也能积极参与，并承担过小组记录员的任务，能够为小组学习贡献力量。平时你能按时完成作业，这说明你有一定的责任心。接下来，如果能在书写规范和遇到难题时的思路整理上继续加强，相信你的学习状态会更加稳定。希望你保持乐于参与、愿意帮助同伴的优点，在新的阶段继续进步。

教师复核清单：
- 请确认"小组记录员"经历是否准确。
- 请确认"作业能按时完成"是否符合本学期整体情况。
- 请确认书写规范和遇到难题急躁的表述是否需要调整语气。
```

## 关键要点

- 每条表述都有记录依据，不凭空编造
- 优点具体化（小组讨论、记录员）而非空话（"积极向上"）
- 不足用"如果可以继续加强"而非"你太差了"
- 结尾积极鼓励，不以问题收尾
- 复核清单逐条让教师核实事実准确性
