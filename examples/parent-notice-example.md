# 示例：将学校通知改写为家长通知

## 输入材料

假设 `02-临时输入/学校通知/科技节活动通知.md` 内容如下：

```markdown
# 关于举办2026年校园科技节的通知

各年级组、各班主任：
为培养学生科技创新精神和实践能力，学校定于5月30日（周五）下午举办校园科技节展示活动。现将有关事项通知如下：
一、活动时间：2026年5月30日14:00-16:30
二、活动地点：学校体育馆
三、参加对象：全校1-6年级学生
四、作品要求：学生需提交一件科技小制作或创意发明作品，可独立完成或小组合作（2-3人）
五、其他事项：请各班主任于5月28日前完成作品收集和登记工作，统一报送至科技教研组。
```

## Trae 调用方式

```
请使用 03-Skills/家长通知改写/SKILL.md。

参考：
- 01-长期资料库/学校要求/家校沟通规范.md
- 01-长期资料库/输出模板/家长通知模板.md
- 01-长期资料库/过往样例/ 中优秀通知样例

处理：
- 02-临时输入/学校通知/科技节活动通知.md

要求：
- 改写成适合发给家长的通知。
- 必须保留原文中的时间、地点、对象、作品要求。
- 不补充原文没有的信息。
- 语气平和、清楚。
```

## 输出示例

```markdown
# 科技节展示活动 家长通知初稿

各位家长好：

学校将于下周五举办校园科技节展示活动，以下是具体安排和需要家长配合的事项。

【通知事项】
校园科技节展示活动

【时间地点】
- 时间：5月30日（周五）下午14:00-16:30
- 地点：学校体育馆

【需要准备】
每个孩子需要提交一件科技小制作或创意发明作品，可以独立完成，也可以2-3人小组合作完成。

【家长需要配合】
请家长提醒孩子在5月28日前将作品带到学校。如果需要帮助准备材料，可以提前和孩子一起计划。

【温馨提醒】
这是孩子们展示动手能力和创造力的好机会，鼓励孩子大胆尝试，重在参与和体验。

---

## 教师复核清单
- [ ] 时间地点是否与学校原文完全一致
- [ ] "5月28日前"的截止时间是否准确
- [ ] 作品数量（每个孩子提交一件）是否正确
- [ ] 是否需要补充作品材料清单或具体要求
```

## 关键要点

- 时间、地点、要求原封不动保留
- 学校内部用语（"各年级组""统一报送"）改写为家长能懂的表述
- 不自行添加活动意义、奖项等原文没有的内容
- 温馨提醒保持鼓励性，不施加压力
